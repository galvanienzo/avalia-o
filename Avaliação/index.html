Introdução

Além de ser uma atividade avaliativa, esse portifólio tem como objetivo armazenar, documentar e ser dos projetos realizados na disciplina Interfaces Web I, lecionada por Luis Felipe Grael Tinós.

Projeto 1: Patrick Estrela

O primeiro projeto, nomeado de Patrick Estrela, apresenta uma biografia da estrela do mar preguiçosa enquanto testa os conceitos de Flexbox

Projeto 2: Cachorros

O segundo projeto, nomeado de Cachorros, apresenta imagens de cachorros fofos e graciosos enquanto testa os conceitos de Responsividade, ideal para dispositivos com exibições distintas.
